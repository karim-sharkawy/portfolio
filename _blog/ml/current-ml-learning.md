# Current ML Learning Journey

I'm currently exploring several cutting-edge areas in machine learning that are shaping the future of AI systems.

## Model Compression & Quantization
Learning how to make large models more efficient without sacrificing performance. This involves techniques like pruning, quantization, and knowledge distillation to reduce model size and inference time by 10x or more.

## Rust for Data Processing
Exploring Rust as a high-performance alternative to Python for data-intensive ML pipelines. The memory safety and zero-cost abstractions make it ideal for building scalable data processing systems.

## Causal Inference
Moving beyond correlation to understand actual cause-and-effect relationships in data. This is crucial for building more robust and interpretable ML systems, especially in areas like A/B testing and decision-making.

## LLM Fine-Tuning
Practical approaches to adapting large language models for specific domains. This includes techniques like LoRA, prompt engineering, and domain-specific fine-tuning to create more accurate and relevant AI applications.

These areas are directly informing my work on production ML systems and helping me stay at the forefront of ML innovation.
