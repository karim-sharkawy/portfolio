# Data Scientist (Volunteer) | Sprout Remote (01/2025-04/2025)

## Overview
In this role, I developed baseline models for mental health intervention planning using topic modeling techniques on social media data.

## Key Responsibilities and Achievements

- **LDA Model Development**: Developed an LDA (Latent Dirichlet Allocation) model on 50k posts to establish a baseline for improving mental health intervention strategies.

- **Model Optimization**: Improved model coherence from 0.344 to 0.432 out of 1 by fine-tuning parameters using grid search techniques.

- **Insight Generation**: Delivered sentiment insights on +75 themes that guided stakeholder outreach and informed decision-making processes.

- **Protocol Development**: Developed accessible data-collection protocols that bridged technical methods with community needs.

## Technologies Used
- Python
- LDA (Topic Modeling)
- Grid Search

## Impact
- Enhanced model coherence by 25%
- Identified 75+ themes for targeted interventions
- Connected technical approaches with community requirements
