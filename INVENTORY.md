## ❓ Common Questions

**Q: How long to deploy?**
A: 30-45 minutes from reading START_HERE to live site

**Q: How do I add a project?**
A: Create `_projects/your-project.md`, use the template, deploy

**Q: How do I write a blog post?**
A: Create `_blog/category/your-post.md`, use the template, deploy

**Q: Can I change colors/fonts?**
A: Yes, edit CSS in `_layouts/default.html` (2-3 minutes)

**Q: How do I add Streamlit demos?**
A: Build Streamlit app, deploy to Streamlit Cloud, link from project

A: Blog monthly, projects quarterly, "Now" monthly
